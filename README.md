# SPACE_MMORPG

Prototype browser-based MMORPG using Node.js and Three.js.

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```
2. Start the server:
   ```
   npm start
   ```
   Then open [http://localhost:3000](http://localhost:3000) in your browser to view a simple rotating ship.

## API Routes

- `POST /auth/login`
- `POST /inventory/update`
- `POST /ship/update`
